# Comic Bot

این پروژه یک ربات تلگرام است که از عکس‌های ارسالی شما یک کمیک با داستان تولید می‌کند.

## طریقه اجرا:

1. پایتون 3.10 یا بالاتر را نصب کنید.
2. کتابخانه‌های مورد نیاز را نصب کنید:

```
pip install python-telegram-bot --upgrade
pip install openai
Pillow
```

3. در فایل `comic_bot.py`، توکن تلگرام و کلید OpenAI خود را جایگزین کنید.
4. اجرای پروژه:

```
python comic_bot.py
```

5. سپس در بات تلگرام /start را بزنید.
